Sea Sketch!
Public Site URL: https://colinfrey.github.io/ColinFrey_Project_3/

Project Concept
Sea Sketch! is an interactive digital canvas inspired by the ocean. This project uses a multi-layered parallax background and fluid animations to create a meditative, artistic environment for drawing.

Key Features and Interaction
Drawing Engine: A custom HTML5 Canvas system that allows for free-form drawing and erasing.

Canvas Customization: Users can toggle and change the background fill color of the drawing card to contrast with their brushstrokes.

History Management: A JavaScript-based undo/redo stack that tracks previous states, allowing for easy creative exploration.

Tool Selection: Includes a brush for drawing, an eraser for corrections, and a one-click clear function with a confirmation modal.

Motion and Design
Parallax Motion: Four distinct CSS-animated background layers moving at different speeds to create a sense of depth and rhythmic movement.

Atmospheric Animation: Uses linear CSS keyframes for a constant, sea-like movement in the background.

Visual Feedback: Interactive buttons use hover effects and a pop-in modal animation to guide user attention through movement.

Technical Execution and Accessibility
WAVE Compliance: Every form control is explicitly linked to a label to pass accessibility standards.

ARIA Support: Icon-only buttons include aria-labels so the site is navigable by screen readers.

Invisible Labels: Used a .sr-only CSS class to maintain a clean visual design while providing necessary context to assistive technologies.

Layout: Built with Flexbox to ensure the toolbar and canvas remain functional on desktop screens.

Assets Used
Fonts: Shrikhand via Google Fonts.

Libraries: normalize.css for browser consistency.

Graphics: Custom-made PNG parallax layers and SVG tool icons.
